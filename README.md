# random-scripts

## Install

Create and activate virtual environment:
```sh
python -m venv .venv
source .venv/bin/activate
```

Install dependencies:
```sh
pip install -r requirements.txt
```

## Usage

Activate the virtual environment:

```sh
source .venv/bin/activate
```

Then, run the script as:

```sh
python <script>.py <args>
```
